## Overview

Freezed is a code generator for immutable classes with a simple syntax and API. It provides:

- Immutable data classes with `copyWith` functionality
- Union types (sealed classes) for type-safe state management
- JSON serialization support via `json_serializable`
- Pattern matching with `when` and `map` methods
- Deep copy for nested objects
- Automatic `toString`, `==`, and `hashCode` generation

### Dart Project

```bash
dart pub add freezed_annotation
```

## Basic Usage

### 1. Define Your Model

```dart
import 'package:freezed_annotation/freezed_annotation.dart';

// Required: associates our file with the code generated by Freezed
part 'main.freezed.dart';

// Optional: for JSON serialization
part 'main.g.dart';

@freezed
class Person with _$Person {
  const factory Person({
    required String firstName,
    required String lastName,
    required int age,
  }) = _Person;

  // Optional: for JSON serialization
  factory Person.fromJson(Map<String, Object?> json) => _$PersonFromJson(json);
}
```

### 2. Run Code Generation

```bash
# One-time generation
dart run build_runner build

# Watch mode (continuously regenerates)
dart run build_runner watch -d

# For Flutter projects
flutter pub run build_runner build
```

## Core Features

### Immutable Data Classes

Freezed automatically generates:
- `toString()` for debugging
- `==` and `hashCode` for value equality
- `copyWith()` for immutable updates

```dart
@freezed
class Person with _$Person {
  const factory Person({
    required String firstName,
    required String lastName,
    required int age,
  }) = _Person;

  factory Person.fromJson(Map<String, Object?> json) => _$PersonFromJson(json);
}
```

**Usage:**

```dart
var person = Person(firstName: 'Remi', lastName: 'Rousselet', age: 24);

print(person); // Person(firstName: Remi, lastName: Rousselet, age: 24)
print(person == Person(firstName: 'Remi', lastName: 'Rousselet', age: 24)); // true
```

---

### copyWith Method

The `copyWith` method allows creating a new instance with specific fields updated:

```dart
var person = Person(firstName: 'Remi', lastName: 'Rousselet', age: 24);

// Update only the name
print(person.copyWith(firstName: 'Dash'));
// Person(firstName: Dash, lastName: Rousselet, age: 24)

// Set age to null (if nullable)
print(person.copyWith(age: null));
// Person(firstName: Remi, lastName: Rousselet, age: null)
```

**Key Points:**
- Only specified fields are updated
- Unspecified fields retain their original values
- Supports setting nullable fields to `null`

### Union Types (Sealed Classes)

Freezed supports union types (sealed classes) for representing multiple states:

```dart
@freezed
sealed class Union with _$Union {
  const factory Union.data(int value) = Data;
  const factory Union.loading() = Loading;
  const factory Union.error([String? message]) = ErrorDetails;
}
```

**Pattern Matching with Switch:**

```dart
var union = Union.data(42);

switch (union) {
  case Data(:final value):
    print('Data: $value');
  case Loading():
    print('Loading...');
  case ErrorDetails(:final message):
    print('Error: $message');
}
```

**Pattern Matching with when:**

```dart
var result = union.when(
  data: (int value) => 'Data: $value',
  loading: () => 'Loading...',
  error: (String? message) => 'Error: $message',
);
print(result); // Data: 42
```

**Pattern Matching with map:**

```dart
var result = union.map(
  data: (Data value) => 'Data ${value.value}',
  loading: (Loading value) => 'Loading...',
  error: (ErrorDetails value) => 'Error: ${value.message}',
);
```

**Difference between `when` and `map`:**
- `when`: Destructures arguments in callback signature
- `map`: Provides full object instance (useful for `copyWith`)

### JSON Serialization

#### Basic JSON Support

```dart
@freezed
class Person with _$Person {
  const factory Person({
    required String firstName,
    required String lastName,
    required int age,
  }) = _Person;

  factory Person.fromJson(Map<String, Object?> json) => _$PersonFromJson(json);
}
```

**Usage:**

```dart
// Serialization
var person = Person(firstName: 'John', lastName: 'Doe', age: 30);
var json = person.toJson();

// Deserialization
var personFromJson = Person.fromJson(json);
```

#### Custom Property Names

```dart
@freezed
class Example with _$Example {
  factory Example(
    @JsonKey(name: 'my_property') String myProperty
  ) = _Example;

  factory Example.fromJson(Map<String, dynamic> json) => _$ExampleFromJson(json);
}
```

#### Union Types with JSON

Freezed automatically uses `runtimeType` field to determine constructor:

```dart
@freezed
sealed class MyResponse with _$MyResponse {
  const factory MyResponse(String a) = MyResponseData;
  const factory MyResponse.special(String a, int b) = MyResponseSpecial;
  const factory MyResponse.error(String message) = MyResponseError;

  factory MyResponse.fromJson(Map<String, dynamic> json) =>
      _$MyResponseFromJson(json);
}
```

**JSON Examples:**

```json
[
  {
    "runtimeType": "default",
    "a": "This uses MyResponse() constructor"
  },
  {
    "runtimeType": "special",
    "a": "This uses MyResponse.special()",
    "b": 42
  },
  {
    "runtimeType": "error",
    "message": "This uses MyResponse.error()"
  }
]
```

#### Customizing Union Key and Value Case

```dart
@Freezed(unionKey: 'type', unionValueCase: FreezedUnionCase.pascal)
sealed class MyResponse with _$MyResponse {
  const factory MyResponse(String a) = MyResponseData;

  @FreezedUnionValue('SpecialCase')
  const factory MyResponse.special(String a, int b) = MyResponseSpecial;

  const factory MyResponse.error(String message) = MyResponseError;

  factory MyResponse.fromJson(Map<String, dynamic> json) =>
      _$MyResponseFromJson(json);
}
```

**JSON with Custom Key:**

```json
[
  {
    "type": "Default",
    "a": "Uses MyResponse()"
  },
  {
    "type": "SpecialCase",
    "a": "Uses MyResponse.special()",
    "b": 42
  },
  {
    "type": "Error",
    "message": "Uses MyResponse.error()"
  }
]
```

## Advanced Features

### Deep Copy

Freezed provides deep copy syntax for nested objects:

```dart
@freezed
class Company with _$Company {
  const factory Company({
    String? name,
    required Director director,
  }) = _Company;
}

@freezed
class Director with _$Director {
  const factory Director({
    String? name,
    Assistant? assistant,
  }) = _Director;
}

@freezed
class Assistant with _$Assistant {
  const factory Assistant({
    String? name,
    int? age,
  }) = _Assistant;
}
```

**Usage:**

```dart
Company company = Company(
  name: 'Google',
  director: Director(
    name: 'John',
    assistant: Assistant(name: 'Jane', age: 25),
  ),
);

// Deep copy - update nested assistant's name
Company newCompany = company.copyWith.director.assistant(name: 'John Smith');

// Update director's name
Company newCompany2 = company.copyWith.director(name: 'Larry');
```

**Null-Safe Deep Copy:**

```dart
// Use ?.call when nested properties might be null
Company? newCompany = company.copyWith.director.assistant?.call(name: 'John');
```

### Default Values

Use `@Default` annotation for default values:

```dart
@freezed
class Example with _$Example {
  const factory Example([
    @Default(42) int value,
    @Default('default') String name,
  ]) = _Example;

  factory Example.fromJson(Map<String, dynamic> json) =>
      _$ExampleFromJson(json);
}
```

**Usage:**

```dart
var example1 = Example(); // value: 42, name: 'default'
var example2 = Example(100); // value: 100, name: 'default'
var example3 = Example(100, 'custom'); // value: 100, name: 'custom'
```

### Mutable Classes with @unfreezed

For mutable data classes, use `@unfreezed`:

```dart
@unfreezed
class Person with _$Person {
  factory Person({
    required String firstName,
    required String lastName,
    required final int age, // Still immutable
  }) = _Person;

  factory Person.fromJson(Map<String, Object?> json) => _$PersonFromJson(json);
}
```

**Usage:**

```dart
var person = Person(firstName: 'John', lastName: 'Smith', age: 42);

// Mutable properties can be changed
person.firstName = 'Mona';
person.lastName = 'Lisa';

// person.age = 50; // ❌ Error: age is final
```

**⚠️ Important Differences:**
- No custom `==` / `hashCode` implementation
- Cannot use `const` constructors
- Collections are modifiable by default

### Custom JSON Converters

For complex JSON deserialization logic:

```dart
class MyResponseConverter implements JsonConverter<MyResponse, Map<String, dynamic>> {
  const MyResponseConverter();

  @override
  MyResponse fromJson(Map<String, dynamic> json) {
    // Already has runtimeType (from serialization)
    if (json['runtimeType'] != null) {
      return MyResponse.fromJson(json);
    }

    // Custom logic to determine constructor
    if (json.containsKey('data')) {
      return MyResponseData.fromJson(json);
    } else if (json.containsKey('specialField')) {
      return MyResponseSpecial.fromJson(json);
    } else if (json.containsKey('message')) {
      return MyResponseError.fromJson(json);
    } else {
      throw Exception('Could not determine constructor for mapping from JSON');
    }
  }

  @override
  Map<String, dynamic> toJson(MyResponse data) => data.toJson();
}
```

**Usage:**

```dart
@freezed
class MyModel with _$MyModel {
  const factory MyModel(
    @MyResponseConverter() MyResponse myResponse
  ) = MyModelData;

  factory MyModel.fromJson(Map<String, dynamic> json) => _$MyModelFromJson(json);
}
```

### Generic Types

```dart
@freezed
sealed class ApiResponse<T> with _$ApiResponse<T> {

```

**Enable in Configuration:**

```dart
@Freezed(genericArgumentFactories: true)
sealed class ApiResponse<T> with _$ApiResponse<T> {
  // ... same as above
}
```

**Or globally in `build.yaml`:**

```yaml
targets:
  $default:
    builders:
      freezed:
        options:
          generic_argument_factories: true
```

**Usage:**

```dart
// Deserialization with type
var response = ApiResponse<User>.fromJson(
  json,
  (obj) => User.fromJson(obj as Map<String, dynamic>),
);
```

## Configuration

### Per-Class Configuration

```dart
@Freezed(
  copyWith: false,      // Disable copyWith generation
  equal: false,         // Disable == and hashCode
  makeCollectionsUnmodifiable: false, // Allow mutable collections
)
class Person with _$Person {
  factory Person(String name, int age) = _Person;
}
```

### Project-Wide Configuration

Create/edit `build.yaml` in project root:

```yaml
targets:
  $default:
    builders:
      freezed:
        options:
          # Format generated files (slower, requires Dart 3.7+)
          format: true

          # Disable features globally
          copy_with: false
          equal: false

          # Union type configuration
          union_key: type
          union_value_case: pascal

          # Generic types
          generic_argument_factories: true

          # Collections
          make_collections_unmodifiable: false
```

## Best Practices

### 1. Use Sealed Classes for State Management

```dart
@freezed
sealed class AuthState with _$AuthState {
  const factory AuthState.initial() = _Initial;
  const factory AuthState.loading() = _Loading;
  const factory AuthState.authenticated(User user) = _Authenticated;
  const factory AuthState.unauthenticated() = _Unauthenticated;
  const factory AuthState.error(String message) = _Error;
}
```

### 2. Prefer Property-Based Pattern Matching

```dart
// ✅ GOOD - Using switch with pattern matching
switch (state) {
  case _Authenticated(:final user):
    return HomePage(user: user);
  case _Loading():
    return LoadingScreen();
  case _Error(:final message):
    return ErrorScreen(message: message);
  default:
    return LoginScreen();
}
```

### 3. Use Shared Properties in Unions

```dart
@freezed
sealed class Example with _$Example {
  const factory Example.person(String name, int age) = Person;
  const factory Example.city(String name, int population) = City;
}

// Shared 'name' property is accessible directly
var example = Example.person('Remi', 24);
print(example.name); // Remi

// copyWith works with shared properties
print(example.copyWith(name: 'Dash'));
// Example.person(name: Dash, age: 24)
```

### 4. Document Your Models

```dart
@freezed
class Person with _$Person {
  const factory Person({
    /// The first name of the person.
    ///
    /// Must not be empty.
    required String firstName,

    /// The person's age in years.
    ///
    /// Must be non-negative.
    required int age,
  }) = _Person;
}
```

### 5. Use @Default for Optional Fields

```dart
@freezed
class Config with _$Config {
  const factory Config({
    @Default(true) bool enableLogging,
    @Default(5) int retryCount,
    @Default([]) List<String> tags,
  }) = _Config;
}
```

## Common Patterns

### Pattern 1: Domain Entity

```dart
@freezed
class User with _$User {
  const factory User({
    required String id,
    required String name,
    required String email,
    DateTime? lastLogin,
  }) = _User;

  factory User.fromJson(Map<String, Object?> json) => _$UserFromJson(json);
}
```

### Pattern 2: API Response DTO

```dart
@freezed
class ApiResponse<T> with _$ApiResponse<T> {
  const factory ApiResponse.success(T data) = Success<T>;
  const factory ApiResponse.failure(String error) = Failure<T>;

  factory ApiResponse.fromJson(
    Map<String, dynamic> json,
    T Function(Object?) fromJsonT,
  ) => _$ApiResponseFromJson(json, fromJsonT);

```

### Pattern 3: Form State

```dart
@freezed
class LoginFormState with _$LoginFormState {
  const factory LoginFormState({
    @Default('') String email,
    @Default('') String password,
    @Default(false) bool isSubmitting,
    @Default(false) bool obscurePassword,
    String? errorMessage,
  }) = _LoginFormState;
}
```

### Pattern 4: BLoC State with Union

```dart
@freezed
sealed class WorkOrderState with _$WorkOrderState {
  const factory WorkOrderState.initial() = _Initial;
  const factory WorkOrderState.loading() = _Loading;

    required List<WorkOrder> workOrders,

  const factory WorkOrderState.error(String message) = _Error;
}
```
