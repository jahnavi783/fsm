## 🚀 How to Use

### **Step 1: Open Project**
1. Launch Android Studio
2. File → Open → Select FSM project folder  
3. Wait for Flutter SDK detection

### **Step 2: Select Configuration**
1. Look for dropdown next to Run button (▶️)
2. Select desired environment:
   - **FSM Development** for local development
   - **FSM Staging** for testing staging APIs
   - **FSM Production** for production testing

### **Step 3: Run**
1. Click Run button (▶️) or Debug button (🐛)  
2. App launches with selected environment configuration
3. Environment automatically detected and configured
